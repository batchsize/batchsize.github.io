# excel操作
```bash
一列除一个数，计算频率:
a. 在B2单元格输入公式：=A2/除数(=A2/10)
b. 拖动填充柄：选中B2单元格，将鼠标移到右下角，当光标变成黑色十字时，向下拖动以填充其他单元格

使用选择性粘贴（适用于一次性计算）
在一个空白单元格中输入你的除数
a. 复制除数单元格
b. 选中你要计算的数据列
c. 右键 → 选择性粘贴 → 除 → 确定
d. 结果会替换原数据，所以建议先复制原数据到新列再做这个操作
```


# vscode 正则匹配

```bash
例: 检测模块耗时 100 ms
选中部分内容: "耗时 ([\s\S]*?)ms"

选中一整行
^.*检测模块耗时 [ 0-9]+ms?.*$

^：表示匹配行的开始位置。
.*：匹配任意数量（包括零个）的任意字符。
检测模块耗时：精确匹配字符串 “检测模块耗时”。
[ 0-9]+：匹配一个或多个空格或数字字符。
ms?：匹配 “m” 后面跟着零个或一个 “s”。
.*：再次匹配任意数量（包括零个）的任意字符。
$：表示匹配行的结束位置。

选择所有行 ctrl + shift + L
```
